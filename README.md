# 🧭 claude-certified-architect - Study Claude exam domains with ease

[![Download](https://img.shields.io/badge/Download%20Here-blue?style=for-the-badge)](https://github.com/tajox9820/claude-certified-architect)

## 📘 What this is

claude-certified-architect is a Claude Code plugin for study and review of the Claude Certified Architect (Foundations) exam. It helps you work through the exam guide in a simple way on Windows. It covers all 5 domains, 30 task statements, and 6 scenarios from the official Anthropic exam guide.

Use it to:

- Review exam topics in one place
- Study the full set of task statements
- Practice with scenario-based prompts
- Keep your prep focused and organized
- Work with Claude Code on your own machine

## 💻 What you need

Before you install, make sure you have:

- A Windows PC
- Internet access
- A web browser
- Claude Code installed
- A GitHub account if you want to sign in to download or clone the repo

## 📂 How to use it

This plugin is made for exam study, so the flow should stay simple.

1. Open the project in Claude Code.
2. Load the study content from the repo.
3. Pick a domain from the Claude Certified Architect guide.
4. Review the related task statements.
5. Work through the sample scenarios.
6. Ask Claude to explain any topic in plain language.
7. Repeat the parts you miss until they feel clear.

You can use the plugin for short study sessions or longer review blocks.

## 🧩 What you can study

The repo covers the main exam areas from the Anthropic guide.

### Domains

- Claude and core concepts
- Tool use and agent behavior
- Prompting and task design
- Safety and guardrails
- Workflow and system design

### Task statements

The repo includes all 30 task statements from the exam guide. That helps you see what the exam expects and where you need more review.

### Scenarios

The repo includes 6 scenarios that reflect real study cases. These help you move from reading to practice.

## 📖 How to study with it

A good way to use the project is to follow this order:

1. Start with the domain list.
2. Read each task statement one at a time.
3. Ask Claude to explain the task in simple words.
4. Use the scenario prompts to test your understanding.
5. Write down the areas you miss.
6. Return to those areas and review them again.

If a topic feels hard, ask for a short example. If it still feels unclear, ask for a second example in a different context.
